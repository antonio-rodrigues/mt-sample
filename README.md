# mt-sample

> VeuJS - Sample app for Android & iOS

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at http://localhost:8084
npm run dev

# serve for android
npm run android

# serve for iSON
npm run ios

# build for production with minification
npm run build

```
